## Requirements

```
1. Java 11.x.y

2. Maven 3.x.y

3. Docker 3.x.y
```

## Setup

### Git Clone

```
https://github.com/lsefiane/publisher-subscriber-microservices.git

```

### Maven Build

```
cd publisher-subscriber-microservices/

mvn clean install

```

### Docker Compose

```
docker-compose --env-file .env up -d

```
